Yotru plugin for WordPress

== Description ==

This plugin adds yotru widget.
Yotru helps your visitors trust you.

For more information visit:

[Yotru](http://yotru.com/)

== Installation ==

1. Upload `yotru` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Api key from Yotru to the settings (Admin > Settings > Yotru)
